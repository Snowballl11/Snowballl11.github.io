P1008 三连击

题目描述

将数字 1 到 9 不重复地分成三组，每组形成一个三位数，要求它们满足比例关系：

$$a : b : c = 1 : 2 : 3 $$

求出所有符合条件的三个三位数，并按第一个数字升序输出。

解题思路

这道题是 枚举 + 字符串判重 的经典问题。核心思路是：

  1. 枚举第一个三位数 a（1 ≤ a < 333），
  2. 计算对应的 b = 2 × a 和 c = 3 × a，
  3. 判断这三个数的所有数字是否刚好用完 1~9 且不重复。

因为 3 × a 最多是 999，所以只需枚举到 332 即可。

  • 本题用它来判断每个数字是否出现。
