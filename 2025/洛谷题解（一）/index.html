P5704 字母转换

题目描述

输入一个小写字母，输出其对应的大写字母。例如输入 q[回车] 时，会输出 Q。

解题

  • 为什么 char 类型的 a 可以 “-32”？

    在 C/C++ 里，char 实际上是一种整数类型（占 1 个字节，通常是 8 位），本质上存储的是一个ASCII 码的数字。

  • char(a - 32) 的作用：

    a - 32 的结果是一个整数类型（整型提升）。只有加上 char()，才会告诉 cout：“把这个数字当作字符的 ASCII 码解释并打印出来。”

P5706 再分肥宅水

题目描述

现在有 t 毫升肥宅快乐水，要平均分给 n 名同学。每名同学需要 2 个杯子。现在想知道每名同学可以获得多少毫升快乐水（结果保留 3 位小数），以及一共需要多少个杯子。

输入格式

输入一个实数 t 和一个整数 n，用空格隔开。

输出格式

输出两行：

  • 第一行输出每个同学可以获得多少毫升快乐水，保留 3 位小数。
  • 第二行输出一共需要多少个杯子。

解题

这道题的关键在于 如何控制小数点后的输出精度，即 setprecision() 函数的使用。

代码实现

知识点讲解

  1. #include <iomanip>
    • 这个头文件包含了控制输入输出格式的函数，比如 setprecision()、setw() 等。
  2. setprecision(n)
    • 控制输出浮点数的小数位数。
    • 默认是“有效数字位数”，但配合 fixed 使用后变为“保留小数点后 n 位”。
  3. fixed
    • 告诉输出流以固定小数点格式显示，而不是科学计数法。
    • 与 setprecision(n) 搭配使用时，输出会精确到小数点后 n 位。

示例说明

输入：

输出：

  • 第一行：500 ÷ 3 = 166.6666...，四舍五入到小数点后三位为 166.667
  • 第二行：每个同学要 2 个杯子，共 3 × 2 = 6

总结

  • 若想控制输出小数位数，务必包含头文件 <iomanip>。
  • 使用 cout << fixed << setprecision(n) 可以稳定地保留 n 位小数。
  • 若不使用 fixed，则会按有效数字位数显示，而不是小数位数。

P5708 三角形面积

题目描述

一个三角形的三边长分别是 a、b、c，那么它的面积为：

$$S = \sqrt{p(p - a)(p - b)(p - c)} $$

其中：

$$p = \frac{1}{2}(a + b + c) $$

输入这三个数，计算三角形的面积，结果保留到小数点后一位。

输入格式

一行输入三个实数 a、b、c，以空格隔开。

输出格式

输出一个实数，表示三角形的面积，保留 1 位小数。

解题

这道题考察 海伦公式 (Heron’s formula) 的应用，以及 <cmath> 库中 sqrt() 函数 的使用。

代码实现

知识点讲解

1. <cmath> 头文件

  • <cmath> 是 C++ 的数学函数库，包含了常用数学运算函数，如：
    • sqrt(x)：平方根
    • pow(x, y)：x 的 y 次方
    • abs(x)：绝对值
    • sin(x)、cos(x)、tan(x)：三角函数
    • log(x)、exp(x)：对数与指数函数

这些函数的参数和返回值一般是 double 类型。

2. sqrt() 函数

  • 功能：计算一个数的平方根。
  • 语法：sqrt(x)
  • 参数：x 必须为非负数，否则结果为 NaN（非数字）。
  • 返回值：返回 double 类型的平方根结果。

  • <cmath> 库提供数学运算支持，是所有数学题常用的头文件。
