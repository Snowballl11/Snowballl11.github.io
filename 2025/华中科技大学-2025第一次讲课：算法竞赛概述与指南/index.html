    什么是算法竞赛，能吃吗？

    主要的算法竞赛/考试包括：

    • 赛氪（7~8月）
    • CSP/CCSP（CSP一年多次，CCSP10月）
    • 天梯赛（4月）
    • 蓝桥杯（初赛4月，复赛6月）
    • 百度之星（初赛暑假，复赛年底）
    • ICPC/CCPC（10~11月）
    • ……

    学这些有什么用？

    • 参加比赛：上至 XCPC，下至睿抗，各种比赛都能充实你的履历，
      锻炼你的意志，功利地说：保研加分。

    • 应付面试：各种算法/数据结构，乃至算法题，都是面试官们最喜
      欢的人股。熟练掌握算法在找工的时候会有一定的好处。

    • 解决实际问题：游戏是如何管理物体碰撞的？关系型数据库如何
      快速有序地存储数据？生产生活中有许多问题要用到算法。

    • 心志的磨练：此路乃修心之路，乃不断斩去心中邪魔，纯正意志，
      无畏坎坷，矢志不渝，求得太一无瑕玲珑心之路！

    想好你的目标

    • 刚进入大学，我想多了解一些团队
    • 如果你对算法竞赛感兴趣，可以接着听听我们后续的萌新入门课程，参与到一些简单的比赛（睿抗/蓝桥杯）
    • CF rating：1200
    • 如果你的目标为睿抗省一/蓝桥省二等，那么萌新入门系列的课程就可以满足了，课上想要完全掌握可能有些压力，需要可以每周做大约5-10题
    • 每个人对于算法竞赛的接受能力差距较大，如果你发现花了很多时间都搞不懂，可以换个赛道（算法竞赛只是计算机很小很小很小的一部分）

    想好你的目标

    • 我想学习一些算法竞赛的知识，参与竞赛丰富经历
      群里我们也会对一些无门槛的算法竞赛进行宣传和通知（天梯赛/百度之星）

    • CF rating：1600
      如果你的目标为天梯赛个人一等奖/蓝桥杯国一/百度之星国二，那么你需要完全掌握萌新培训的内容，自学提高组以内的算法（洛谷绿题以内）；每周需要做10-15题，总题量约1000题

    • 如果你的目标是保研加分，那建议可以到此为止了（如果你有CF 1600分的水平，那么大概可以获得 CSP350+蓝桥杯国二+百度之星国二+天梯赛个人二等奖+睿抗国一，已经可以把保研加分凑满了）

    ICPC/CCPC 赛事介绍

    • 与其他大多数算法竞赛不同，XCPC 采用三人一队的形式参加，但是只有一台电脑可供解题
    • 今年 ICPC 共有 7 个赛站，华科预计有 21 队次参加；CCPC 有四个赛站，预计有 9 个队次参加。
    • 共计 5 小时，一般有 12~13 题
    • 每道题只有通过与不通过之分，成绩先按照通过数排序，再按照罚时排序
    • 罚时为每道最终通过的题的（通过时间 + 尝试次数 * 20）之和

    如何获得 ICPC/CCPC 名额

    • 对于新生，可以参加10月举办的新生名额选拔赛，比赛将会分配一个 CCPC 和一个 ICPC 名额，
      欢迎大家留意后续通知
    • 对于高年级同学，需要在暑假参加牛客多校/杭电多校和9月的网络预选赛，绝大多数的名额将会按照以上比赛的成绩进行分配（CF：1900+）
    • 如果你现在是零基础同学，除非你对算法竞赛有极大的热情并且愿意付出至少每周10小时的时间，否则根据经验很难得到名额

    CPP 语法基础介绍

    C 和 CPP 的区别

    • CPP（C++，C Plus Pluse）是 C 的超究极威力无敌加强版。
    • CPP 完美兼容 C。
    • 算竞基本上使用的是 CPP，虽然最常用的是 C 里面就有的一些特性。

    • cin/cout 是最方便的 IO 工具。
    • cin 后方接上 >> ，后方跟上你希望读入的变量。
    • cout 后方 << ，然后接上你希望输出的内容。

    变量类型

    现在总该认识变量了

    变量的生命周期

    • 一个变量的声明包含类型说明符、变量名和初始值。
    • 一个局部变量的作用域是从定义点开始到它所在的代码块结束。
    • 在一个变量的作用域内，它可以被访问和修改。
    • CPP 中，等号的意义是赋值，双等号的意义才是判等。
    • 当变量所在的作用域结束时，变量会被销毁。

    常量

    • 当以变量的方式声明的时候前方加上 const 就变成了常量。
    • 常量必须有初始值；常量不能被更改。
    • 可以防止很多傻逼错误。

    比较和逻辑运算符

    比较运算符：

    • > 大于
    • >= 大于等于
    • < 小于
    • <= 小于等于
    • == 等于
    • != 不等于

    逻辑运算符：

    • && 逻辑与
    • || 逻辑或
    • ! 逻辑非

    二进制、位运算

    • 整型的二进制表示：
      6 -> 110, 11 -> 1011
    • 按位
    • 按位与：1&0=0, 1&1=1, 0&0=0
    • 按位或：1|0=1, 1|1=1, 0|0=0
    • 按位异或：1^1=0, 0^0=0, 1^0=1
    • 按位异或可看作不进退位的加减法。

    常见位运算符：

    • ~ 按位非
    • & 按位与
    • | 按位或
    • ^ 按位异或
    • << 按位左移
    • >> 按位右移
