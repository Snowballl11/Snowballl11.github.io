她

Posted on 2024-01-22

以此文献给我的12至18岁。

我和她认识是在17年末，如果没记错是在10月。那时候我初一，社交圈子基本都是熟人，她算是我最早认识的的一批网友，现在是唯一一个还在聊天的网友^1。那时候我获取信息的途径主要还是从老师、家长和同学那里。当时从跟我最早认识的那批网友的聊天，给我带来了相当大的震撼。

那之前，我对游戏的看法还都是听家长、老师说，结果自然是“游戏都是洪水猛兽”、“百害无一利”这样，那时候即使也在玩游戏，却是很有负罪感，都是偷着玩的；我对日本动画的看法，也都来自于我的家长、老师，就是“日本鬼子搞出来荼毒中国小孩的东西，滚出中国”。那之后，我才从最早那批网友的空间和消息里意识到^2，原来玩游戏和看动画是正常的娱乐，也是可以拿出来说的。

那时候，我的世界观开始有了一些变化，逐渐意识到这个世界好像和身边人说的不太一样。

因为看动画嘛，我也接触了B站。从一些UP的视频中，我更意识到这个世界跟我想的不一样，尽然有些视频的观点过于偏激或者是错误的，但这些仍给了我一个新的角度来看待世界。我无法想像，如果当时没有遇到她，现在的我又是什么样。但敢肯定，我一定更喜欢现在的自己。

这中间还发生不少事，我们一起玩游戏、喊我去看台风^3、与父母发生矛盾时她为我排忧解难、中考的时候给我送上祝福……^4

从12到18，她陪我度过了我的整个青年时代，不知不觉中她也成为了我这个人生阶段中最重要的人。

她一共救赎过我两次，第一次是我们认识的时候，她把我从只会听父母老师的话的无趣人生中解救出来，第二次是她离开的时候，她把我从毫无希望、自我怀疑的最低谷解救出来。

这些年她也变了许多，也再也不是我要找的那个慕瞳了，如果我要找慕瞳那我就不应该来找小柯。

最后，她删了我的好友，或许是她觉得她不能再听我跟她倾诉了，或许是她不再把我当作她的朋友了。我的解释是她陪我走过了从12到18，现在她想告诉我“你成年了，以后的路没我你自己一样能走好”。这一切就像一场梦，梦醒了…就该往前走了。

软件更新

  • 使用Software Updater更新

    Ubuntu的Software Updater（软件更新器）是一个图形化的工具，用于管理系统中的软件更新。它提供了一种简单方便的方式来检查可用的软件更新并安装它们。

  • 使用sudo apt update && sudo apt upgrade命令更新

    sudo apt update && sudo apt upgrade命令的作用是先更新软件包列表，然后升级系统上已安装的软件包。

  • 区别：

使用Software Updater和使用sudo apt update && sudo apt upgrade命令更新系统存在一些区别：

  1. 用户界面 vs 命令行：Software Updater提供了一个图形化的用户界面，适合那些更喜欢使用可视化工具的用户。它显示可用的更新，并允许用户选择要安装的更新。而sudo apt update && sudo apt upgrade命令是在命令行中执行的，适合那些喜欢使用命令行界面的用户。
  2. 更新范围：Software Updater关注整个系统的更新，包括操作系统、内核、系统组件和已安装的软件包。它会检查并提供相关的更新选项。而sudo apt update && sudo apt upgrade命令主要用于更新已安装的软件包。
  3. 灵活性和控制：使用命令行更新命令可以提供更多的灵活性和控制。你可以根据需要执行不同的apt命令，如安装特定软件包、删除软件包等。而Software Updater主要提供了一个简单的界面，用于更新系统和软件包，但对于更高级的软件包管理操作，可能需要使用命令行。
  4. 自动化：Software Updater可以设置为自动检查和安装更新，使得系统保持最新状态更加方便。而使用命令行更新需要手动执行相应的命令。

综上所述，Software Updater提供了一个简单易用的图形化界面，适合普通用户快速更新系统和软件包。而使用sudo apt update && sudo apt upgrade命令则提供更多的灵活性和控制，适合那些习惯于使用命令行的用户或需要进行更高级的软件包管理操作的用户。

AI

Posted on 2023-06-14 Edited on 2024-01-22

根据是否模拟人类神经网络，人工智能可以分为两类：符号人工智能 和 连接主义人工智能。

符号人工智能是基于符号逻辑和规则的人工智能，它使用人工设计的符号和规则来表示和处理知识和推理。这种方法常用于专家系统和推理引擎中，但它的缺点是需要大量的人工设计和编程，不能很好地处理模糊和不确定性的情况。

连接主义人工智能是基于神经网络的人工智能，它模拟人类神经元之间的连接和交互，从而实现自我学习和自我适应。这种方法可以很好地处理模糊和不确定性的情况，但需要大量的数据和计算资源来训练和优化神经网络。

人工智能的意识问题

对于人工智能的意识的问题，我倾向于将人工智能分为“符号人工智能”和“连接主义人工智能”来研究。后者依靠模拟人类神经元之间的连接和交互从而产生“智能”，这样的方式让我感觉它比前者更有可能产生我们人类通常认知中的“智能”。

这并不意味着前一种人工智能久一定不会具有意识，作为人类我们会天然的认为基于“神经元连接”产生意识的方式，有别于符号人工智能基于符号逻辑和规则产生智能的方式，但这很有可能是人类的一种不经思考的自大。符号人工智能的符号逻辑和规则和我们的神经元所完成的功能，是否有本质区别呢？

  1. 在一定意义上神经元完成的工作可以用简单的语言概括，而符号人工智能的符号逻辑和规则可以被看作是一种抽象和形式化的表达方式，它们可以被用来描述和模拟各种现实世界中的问题和过程，我们将这样的问题和过程向神经元的功能的方向靠拢，这样的符号人工智能是不是能称为连接主义人工智能？这样来看，符号人工智能的符号逻辑和规则和我们的神经元似乎没有本质区别。

  2. 另一方面，符号逻辑和规则是由人工设计和编程的，是一种人工智能的表达方式，它是基于一些预先定义好的符号和规则，来进行推理和决策的；神经元则是自然进化而来的，它们能够自适应地学习和处理各种信息，并将其转化为电信号进行传递和处理。前者的功能是确定且一成不变的，后者在自然界中不断进化，它的功能也在进化中缓慢改变，这似乎能表明符号人工智能和神经元之间存在本质区别。

如果再考虑到机器“自编程”，这似乎也能模仿人类神经元的进化，这无疑使我们的探讨更加复杂了。总之对于我们所探究的两类人工智能，我们都无法否认其产生“意识”的可能。

人工智能的终点在哪

我们常用“强弱”来评价一个人工智能产品的水平如何，那人工智能能够无限的变“强”吗？我希望用AlphaGo和当下火热的ChatGPT来阐述我的观点。

我认为AlphaGo和ChatGPT是完全不同的两类AI，AlphaGo面向围棋的AI, 围棋的规则是固定的，你下的每一轮围棋都在相同且完备的规则下进行；ChatGPT是面向现实世界的AI，他追求的不是棋盘上博弈的胜利，而是要服务于一个物理法则都没有被完全探究清楚的世界（指对于微观世界规律的探索无穷无尽，我们几乎无法完全地探索清楚微观世界的奥秘）

讲到这里你或许疑问这两者的本质区别到底在哪里，这是很好的问题，接下来我要引用毛泽东文章里的一段话，来解释这个问题：

人的正确思想是从哪里来的？是从天上掉下来的吗？不是。是自己头脑里固有的吗？不是。人的正确思想，只能从社会实践中来，只能从社会的生产斗争、阶级斗争和科学实验这三项实践中来。

对于这段话，我们完全可以去掉人的成分，而它依然成立：正确思想是从哪里来的？是从天上掉下来的吗？不是。是自己头脑里固有的吗？不是。正确思想，只能从社会实践中来。

对于AlphaGo而言，它面相的对象是围棋，围棋有固定的规则，它不需要再对规则做探索，它的每次与人或非人的博弈，或学习优秀棋局数据的行为，对AlphaGo来讲都是一次实践，这也是AlphaGo为什么能在不再与人博弈之后，依然通过自我博弈不断提升的原因。AlphaGo在不再与人类对局之后，通过不断地自我博弈，围棋水平不断增强。

而对于ChatGPT而言，它面向的对象是现实世界，但ChatGPT是大语言模型（LLM），他对世界的正确认识来源于它所学习的语言数据，这些语言数据之所以能携带正确的认知，归根究底来源于人类的社会实践活动，没有人类的社会实践活动，也就不可能有ChatGPT正确的认知。这也正解释了为什么有研究表明：用AI产生的语料来训练AI将使其退化并最终崩溃^1,在AI使用自己产生的预料自我训练的时候，并没有经历社会实践的过程，其正确的认知（思想）并没有增加，但在这样的情况下仍然以自己产生的语料作为训练的素材，最终导致了其退化并最终崩溃。

可以这样说，AlphaGo和ChatGPT的本质区别在于，AlphaGo能够通过自我博弈完成实践活动，而ChatGPT无法仅通过自身完成实践活动，它的全部正确思想来源于人类语料，而它无法从自身继续获取正确的思想，所以若想增强ChatGPT，则需要向其输入更多人类语料，而若想实现AlphaGo的增强，则完全不需依附于人类实践。

机器能否进行类似人类的实践活动呢？能让机器自己从实践中获取正确思想吗？仅从技术向讲，这是可以实现的，但这同时需要面临伦理问题。
