嘿. 这里有一些经验能让你更舒服地使用Windows PC.

1.软件#

本文提到的所有软件均可在这里下载.

1.1系统软件#

现在笔记本出厂时携带的系统一般是家庭版, 家庭版系统会限制一些功能或组件(如Hyper-V), 因此不推荐计算机及相关专业的同学使用. 建议在拿到电脑之后, 立即使用最新的消费者版(consumer editions)系统镜像(.iso)进行重装, 并在引导界面选择专业工作站版安装.

1.2应用软件#

小心流氓软件, 在安装(和卸载)软件的过程中一定要格外留意安装程序中的复选框, 有下拉菜单要展开仔细检查, 切勿「快速安装」.

1.2.1通用程序#

  • 压缩解压软件: 7-zip
  • 浏览器: Google Chrome
  • 下载器: IDM
  • 播放器: PortPlayer / vlc
  • 文本编辑器: VScode
  • 办公软件: office / Microsoft 365
  • PDF查看器: Adobe Acrobat Pro DC
  • 应用卸载工具: geek uninstaller

1.2.2专业程序#

  • 专业排版工具: TexLive
  • 版本管理工具: Git / Github Desktop
  • 虚拟机工具: VMware Workstation Pro
  • 远程(ssh/ftp)工具: xshell & xftp

1.2.3系统程序#

  • Windows Powershell
  • 控制面板
  • 磁盘管理
  • 注册表 / 组策略 / …

1.2.4可选/娱乐程序#

  • 密码管理工具: Authy(Mobile)
  • 电子书阅读器: calibre
  • 局域网传输工具: LocalSend
  • 录屏直播推流工具: OBS
  • BT下载工具: qBittorrent / BitComet(安装程序含有推广)

2.习惯#

2.1系统设置#

  1. 激活: Microsoft-Activation-Scripts
  2. 文件资源管理器: 设置显示隐藏的项目和显示文件扩展名
  3. 地区: 建议设置美国
  4. 输入法: 建议添加「微软拼音」和「英语(美国)」两款键盘
  5. 配置电源选项
  6. 设置开始菜单,任务栏

2.2使用技巧#

  • 应用卡死强制停止: Control + Shift + Esc
  • 使用火绒代替Windows Defender
  • 甄别假网站和各类无良「软件园」「下载站」
  • U盘使用: 分区表建议exFAT
  • 规范目录结构
